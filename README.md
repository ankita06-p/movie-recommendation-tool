# Movie Recommendation Tool 🎞️

## Description
A Machine Learning project that recommends top 5 movies most similar to the user’s favourite movie by content-based filtering using Python, TfidfVectorizer, and cosine similarity. The system analyses movie features such as genres, keywords, and overviews to find similar movies and generate personalized recommendations.

## Dataset
Download the dataset from:
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata/data

## Features
- Recommends similar movies
- Uses cosine similarity
- Uses movie metadata like genre, keywords, and director
- Beginner-friendly ML project

## Technologies used
- Python
- Pandas
- NumPy
- scikit-learn

## Example
### Input-
The Godfather
### Output-
The Godfather: Part II  
The Godfather: Part III  
The Cotton Club  
Point Blank  
Blood Ties
